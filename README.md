# Bright Future Secondary School Website

A modern, fully responsive website for **Bright Future Secondary School**, Onitsha, Anambra State, Nigeria.

## 🌐 Live Site
> Deploy on Netlify or GitHub Pages

## 📁 Project Structure

```
my-first-project/
├── module-11/
│   ├── index.html      ← Main website (all-in-one)
│   ├── script.js       ← Interactions & animations
│   └── school.json     ← School configuration data
├── README.md
├── LICENSE
└── package.json
```

## ✨ Features

- **Responsive design** — works on mobile, tablet, and desktop
- **Sticky navbar** with active link highlighting and mobile hamburger menu
- **Hero section** with school stats
- **Why Choose Us** — 6 feature cards with hover animations
- **About section** with image and key facts
- **Academics** — subjects offered in a clean grid
- **Admissions** — step-by-step guide + key dates card
- **News & Announcements** section
- **Contact section** with form and contact details
- **Scroll reveal animations** on all sections
- **Back to top** button
- **Footer** with quick links and social icons

## 🎨 Design

- **Font:** Playfair Display (headings) + Plus Jakarta Sans (body)
- **Colors:** Navy `#1E3A8A` + Gold `#F59E0B`
- **Theme:** Professional, clean, education-focused

## 🚀 Deployment

### Netlify
1. Push to GitHub
2. Connect repo on [netlify.com](https://netlify.com)
3. Set **Publish directory** to `module-11`
4. Deploy

### GitHub Pages
1. Go to repo **Settings → Pages**
2. Set source to `main` branch, `/module-11` folder
3. Save — site goes live at `https://username.github.io/my-first-project`
